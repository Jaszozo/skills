# 社交媒体案例研究助手

帮助用户高效完成小红书、抖音等平台的竞品分析和案例收集工作。

## 核心价值

将原本需要 **6-7小时** 的社媒研究工作压缩到 **1-1.5小时**。

| 步骤 | 传统方式 | 使用本工具 |
|-----|---------|-----------|
| 确定研究对象 | 1小时 | 0分钟 (Claude完成) |
| 漫无目的刷社媒 | 2-3小时 | 0分钟 |
| 有针对性截图 | - | 30-45分钟 |
| 整理分类 | 1小时 | 0分钟 (自动创建文件夹) |
| 写分析报告 | 2小时 | 30分钟 (Claude辅助) |

## 使用方法

### 1. 启动研究

告诉Claude你想研究什么：

```
帮我研究瑞幸咖啡的竞品在小红书上的营销活动
```

或者：

```
我想收集新茶饮品牌的联名案例，准备做竞品分析PPT
```

### 2. Claude会帮你

1. **搜索行业信息** - 找出竞品清单、热门活动
2. **创建文件夹** - 在桌面创建整理好的文件夹结构
3. **生成搜索词** - 告诉你该搜什么

### 3. 你只需要

1. 打开桌面上的 `截图清单.md`
2. 按里面的搜索词去小红书/抖音搜索
3. 截图后拖入对应文件夹
4. 回来告诉Claude "截图完成"

### 4. 获得报告

Claude会分析你的截图，生成：
- 完整的案例分析报告
- 跨品牌对比表格
- 关键洞察总结
- (可选) PPT大纲

## 技术说明

### 为什么不能自动截图？

小红书、抖音等平台：
- 需要登录才能查看完整内容
- 有反爬机制
- 内容动态加载

因此采用"研究辅助+手动收集"的混合模式，既高效又可靠。

### Claude能做什么？

| 能力 | 说明 |
|-----|------|
| Web搜索 | 搜索行业报告、案例分析文章 |
| 创建文件夹 | 在桌面自动创建项目结构 |
| 读取图片 | 分析用户提供的截图 |
| 生成报告 | 整理分析、生成报告和PPT大纲 |

### Claude不能做什么？

| 限制 | 原因 |
|-----|------|
| 直接访问小红书/抖音 | 需登录、有反爬 |
| 自动截图 | 需要浏览器自动化环境 |
| 实时抓取内容 | 平台限制 |
