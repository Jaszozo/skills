# 商业爆发式增长推理 Skill - 使用说明

## 📦 文件说明

### 1. **SKILL.md** - 核心框架文档
这是skill的主文件，包含完整的"道法术"分析框架和使用指南。

**核心内容：**
- 【道】借势与痛点：势能分析（点线面体）+ 痛点挖掘
- 【法】战略设计：差异化定位 + 价值主张 + 竞争壁垒
- 【术】增长策略：冷启动 + 规模化 + 增长飞轮
- 使用指南：事后分析 vs 事前推理的不同用法

**关键特点：**
- 输出格式：bullet point简洁呈现
- 核心逻辑：借势 → 痛点 → 解决方案
- 痛点与解决方案一一对应
- 增长飞轮可灵活省略

### 2. **example-yuanqisenlin.md** - 元气森林案例（事后分析）
展示如何使用框架拆解成功品牌的增长逻辑。

**分析重点：**
- 元气森林借了什么势（健康消费升级 + 减糖风潮）
- 挖掘了什么痛点（年轻人想喝碳酸饮料但怕胖）
- 提供了什么解决方案（代糖技术 + 0糖0脂）
- 一句话总结底层逻辑

### 3. **example-new-brand.md** - 新品牌推理案例（事前推理）
展示如何使用框架为新品牌规划增长策略。

**推理重点：**
- 分析可借什么势（女性悦己消费 + 情绪价值爆发）
- 挖掘什么痛点（职场女性需要自我关爱情绪出口）
- 设计什么解决方案（治愈系咖啡品牌）
- 包含关键假设验证清单

## 📖 框架核心理念

### 成功品牌的底层逻辑
**借大势 → 找到痛点 → 放大痛点 → 提供解决方案**

### 点线面体模型
- **体**（长趋势）：消费心智、社会认知
- **面**（短趋势）：突发事件、行业危机
- **线**（赛道）：市场机会点
- **点**（企业）：独特优势

### 使用场景
1. **事后分析**：拆解成功品牌做对了什么
2. **事前推理**：为新品牌设计增长策略
3. **针对消费品牌**的市场分析

## ✅ 输出原则

- **简洁至上**：只留能推导出成功模式的信息
- **逻辑通顺**：借势→痛点→解决方案形成完整链条
- **一一对应**：痛点与解决方案要有清晰对应
- **重点清晰**：用bullet point呈现
- **灵活省略**：无明显飞轮可不写
